# Biblioteca Online

## Descriere generală

**Biblioteca Online** este o aplicație web care digitalizează funcționalitățile unei biblioteci clasice, oferind utilizatorilor posibilitatea de a descoperi, împrumuta și evalua cărți într-un mod simplu și eficient. Aplicația este concepută pentru a oferi o experiență modernă, intuitivă și sigură, atât pentru utilizatori, cât și pentru administratori.

## Funcționalități

### Acces Utilizator și Administrator
- Autentificare și înregistrare 
- Roluri diferite pentru utilizatori și administratori

### Gestionare Cărți
- Vizualizarea listei de cărți disponibile
- Căutare și filtrare după titlu, autor sau categorie
- Vizualizarea detaliilor complete ale unei cărți
- Sortarea alfabetica dupa titlu sau autor

### Recenzii și Evaluări
- Utilizatorii pot lăsa stelute pentru cărțile împrumutate

### Împrumuturi și Favorite
- Listă de cărți favorite (wishlist)
- Evidență a disponibilității cărților

### Management Dinamic
- Administratorii pot adăuga, edita și șterge cărți
- Vizualizarea utilizatorilor 

## Tehnologii utilizate
**Backend:** Spring Boot pentru dezvoltarea unor aplicații server-side robuste și scalabile.  
**Frontend:** React.js pentru o interfață de utilizator dinamică, modernă și responsivă.  
**Bază de date:** MySQL pentru stocarea și gestionarea eficientă a datelor.  
**Securitate:** Spring Security pentru autentificare, autorizare și protejarea datelor utilizatorilor.

## Concluzie
**Biblioteca Online** este o aplicație completă care oferă o soluție modernă pentru gestionarea unei biblioteci digitale. Proiectul este potrivit pentru uz educațional și poate fi extins cu funcționalități suplimentare în viitor.
